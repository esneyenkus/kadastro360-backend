# Parsel & Eğim — Kurulum

## 1. Node.js kur (eğer yoksa)
https://nodejs.org → LTS sürümünü indir ve kur

## 2. Klasöre gir ve server'ı başlat
Terminali (CMD veya PowerShell) aç:
```
cd C:\Users\KULLANICI_ADIN\Desktop\parsel_app
node server.js
```
Terminalde şunu görmelisin:
```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  Parsel & Eğim Proxy Server
  http://localhost:3000
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

## 3. index.html'i aç
Aynı klasördeki index.html dosyasına çift tıkla.

## Kullanım
İl → İlçe → Mahalle seç → Ada ve Parsel no gir → Sorgula
